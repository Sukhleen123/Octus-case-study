# Financial Multi-Agent Intelligence System

## Overview

This repo implements a **Financial Multi-Agent Intelligence System** that answers natural-language questions over:

- **Octus unstructured documents**: 15 companies, 105 earnings call transcripts, 137 SEC filings (10-K / 10-Q) with full HTML content
- **SimFin structured financial statements**: income, balance sheet, cash flow — fetched via the SimFin v3 API

The system features:
- **LangGraph multi-agent pipeline**: router → doc agent + SimFin agent (parallel) → synthesis
- **Semantic document retrieval**: Pinecone vector search over two namespaces (transcripts, SEC filings)
- **Structured financial queries**: DuckDB over 6 pre-ingested SimFin tables
- **Inline citation system**: every answer references numbered sources; Octus chunks are expandable, SimFin data renders as tables

## Quick Start

### 1. Install dependencies

```bash
pip install -e ".[dev]"
```

### 2. Configure environment

```bash
cp .env.example .env
# Edit .env — minimum required for full functionality:
#   ANTHROPIC_API_KEY=...
#   SIMFIN_API_KEY=...
#   PINECONE_API_KEY=... (optional; FAISS fallback used if absent)
```

### 3. Run ingestion (once)

```bash
python -m src.ingest
```

This populates all data artifacts:
1. Parse and normalize Octus JSON/HTML → `data/processed/octus/documents.parquet`
2. Chunk transcripts (RecursiveChunker) and SEC filings (SECSectionChunker)
3. Embed chunks and index into Pinecone (or FAISS fallback)
4. Match Octus companies to SimFin tickers (rapidfuzz fuzzy matching)
5. Fetch SimFin financial statements via v3 API → `data/processed/simfin/simfin.duckdb`

Partial re-runs are supported:

```bash
python -m src.ingest --octus-only   # Re-index documents only
python -m src.ingest --simfin-only  # Re-pull SimFin data only
```

### 4. Run the app

```bash
streamlit run app/streamlit_app.py
```

### 5. Run tests

```bash
pytest tests/ -v
```

## Key Settings

All settings are read from environment variables (or `.env`):

| Setting | Default | Purpose |
|---------|---------|---------|
| `ANTHROPIC_API_KEY` | — | Required for LLM synthesis and doc agent retry |
| `SIMFIN_API_KEY` | — | Required for SimFin financial data ingestion |
| `PINECONE_API_KEY` | — | Optional; FAISS fallback used if absent |
| `EMBEDDING_PROVIDER` | `sentence_transformers` | `sentence_transformers` / `openai` / `anthropic` |
| `VECTORSTORE_PROVIDER` | `auto` | `auto` (Pinecone if key set, else FAISS) |
| `RETRIEVER_ID` | `dense` | `dense` or `dense_mmr` |
| `MAPPING_MODE` | `auto_matched` | Which Octus→SimFin mappings to include |
| `LLM_PROVIDER` | `anthropic` | `anthropic` or `none` (disables synthesis) |
| `DOC_AGENT_MODEL` | `claude-haiku-4-5-20251001` | Model used for doc agent sparse-result retry |
| `TABLE_FORMAT` | `parquet` | `parquet` or `csv` for output tables |
